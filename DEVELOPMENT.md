# Pull down api

Use `rake` to pulldown api information.

    rake pull:starships
    rake pull:people
    rake pull:vehicles

To removed downloaded file use

    rake clear

# Install dependencies and run the application

```
yarn install
yarn elm-app install
yarn elm-app start
```

or (caution: use _npx_ not _npm_)

```
npx install
npx elm-app install
npx elm-app start
```
